# Governance

## Project Structure

This project is a Rust port of [simonnilsson/ios-uploader](https://github.com/simonnilsson/ios-uploader).

## Decision Making

- Major changes require discussion in GitHub issues
- Pull requests require review before merging
- Security issues are handled privately

## Release Process

1. Update version in Cargo.toml
2. Update CHANGELOG.md
3. Create a GitHub release
4. CI/CD automatically builds and uploads binaries

## Communication

- GitHub Issues for bugs and feature requests
- GitHub Discussions for questions and ideas
- Pull Requests for code contributions

## Code of Conduct

Please be respectful and constructive in all interactions.
